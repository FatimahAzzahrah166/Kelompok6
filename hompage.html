HTML Language

Pengertian HTML

HyperText Markup Language (HTML) merupakan suatu bahasa markah yang digunakan untuk membuat halaman web, menampilkan berbagai informasi di dalam sebuah browser web. HTML dibuat dengan menggunakan aplikasi pengolah kata dan disimpan dalam format ASCII sehingga menjadi halaman web dengan perintah-perintah HTML.

HTML bukanlah bahasa pemrograman, dan itu berarti HTML tidak punya kemampuan untuk membuat fungsionalitas yang dinamis. Sebagai gantinya, HTML memungkinkan user untuk mengorganisir dan memformat dokumen, sama seperti Microsoft Word.

Fungsi HTML

1. Memudahkan Penggunanya

2. Menandai Text Pada Suatu Laman

3. Sebagai Dasar Bahasa Website

4. Untuk Menampilkan Tabel, Gambar, Video dan Lainnya

5. Menandai Elemen Table dan Membuat Online Form
